Dependencies for NFT Demo

Node v12.18.2 or later
Any Code Editor
Truffle
Ganach
Infura ID
Pinnata Account


1. Clone the repo
2. npm init -y
3. update .env with your credentials
     i. create account on pinnata to get API Key and Secret key
     ii Update mnemonic from your metamask
     iii. Use infura rinkeby URL

Migrate to rinkeby network
Get the contract id
Go to https://bridge.mintnft.today/
Access the deployed nft using IPFS url and IPFS hash from the ipfsHash.json
